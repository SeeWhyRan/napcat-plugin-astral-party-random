# napcat-plugin-astral-party-random

NapCat 插件：星引擎派对随机开局生成器。

本插件用于在群聊/私聊中根据预设随机生成“开局方案”（地图、难度、角色筛选结果等），并提供 WebUI 管理界面以便维护配置、查看状态与群启用情况。

插件显示名：`星趴随机生成`

## 功能

- 随机开局：一条命令按“当前选择的预设”生成结果
- 预设系统：支持全局预设（管理员配置）与个人预设（用户导入）
- 选择记忆：用户可设置默认使用的预设（全局/个人 + 序号）
- 群开关：可按群启用/禁用（WebUI 批量管理）
- WebUI：查看状态、修改配置、管理群启用、管理预设数据（通过接口）

## 运行环境

- NapCat

## 使用说明

### 1) 随机开局

- `/随机开局`

行为说明：

- 若你没有设置默认预设：优先使用“全局 1”；如果没有全局预设，则使用“个人 1”
- 若你已设置默认预设：按你的选择生成

### 2) 帮助

- `/随机开局帮助`

### 3) 预设管理

查看预设列表：

- `/随机开局预设`

导入个人预设（两步流程）：

1. 发送：`/随机开局导入预设`
2. 在 3 分钟内发送预设 JSON 文本（从网站导出的 JSON 文件中复制内容即可）
3. 再发送备注名（用于列表展示）
4. 期间可发送 `/取消` 退出

删除个人预设：

- `/随机开局删除预设<数字>`

设置默认预设：

- `/随机开局设置[全局|个人][数字]`

示例：

- `/随机开局设置全局2`
- `/随机开局设置个人1`

说明：

- “全局预设”只能在插件配置中维护（`globalPresets`）
- “个人预设”由用户自行导入，按 QQ 号分别存储

### 4) 预设来源（配置生成网站）

导入前请先在以下网站配置并导出预设：

- https://ap.xypp.cc/

导出后会下载一个 JSON 文件，打开并复制其中文本发送给机器人即可。
